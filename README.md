# Study Planner Web Service

This project is a simple web-based study planner for students. It was created for Introduction to AI Programming Assignment 4.

## Project Overview

Study Planner helps students organize study tasks by subject and priority. Users can add tasks, mark them as completed, delete tasks, filter tasks by priority, and check their study progress.

## Main Features

- Add study tasks with title, subject, and priority.
- Display tasks in a clear task list.
- Mark tasks as completed.
- Delete tasks.
- Filter tasks by priority.
- Automatically calculate completion progress.
- Save tasks in the browser using localStorage.
- Responsive layout for different screen sizes.

## Page Structure

The website has three main sections:

1. Home
2. Planner
3. Study Tips

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub
- Vercel

## How to Run Locally

1. Download or clone this repository.
2. Open `index.html` in a web browser.
3. Add, complete, delete, and filter study tasks.

No backend server or database is required.
